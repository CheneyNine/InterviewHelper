# ADR-0002：输出可观察的训练指标，不进行情绪诊断

## Status

Accepted

## Context

面部、声音和姿态会受文化、设备、健康状态和个人习惯影响。将其直接解释为焦虑、不自信或不诚实，缺少可靠依据并带来隐私、偏差与就业使用风险。

## Decision

MVP 只输出语速、停顿、填充词、脸部是否在画面中等可观察指标，以及带时间证据的表达建议。输出不得用于自动招聘决策，不产生医学、心理或人格标签。

## Consequences

### Positive

- 输出更容易验证、解释和纠错。
- 降低对用户造成误导和歧视的风险。

### Negative

- 产品文案不能宣称准确识别紧张或焦虑。
- 需要设计更细致的中性反馈语言。

## Alternatives Considered

- 直接让全模态模型给出情绪百分比：实现容易，但无法校准且不可解释，因此拒绝。
