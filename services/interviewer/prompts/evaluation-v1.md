# Answer Evaluation Prompt v1

综合评估接收题目、题目级参考答案、题目级评分标准、回答文本和 Multimodal report。

## 固定评分结构

- 内容质量：70%
  - 相关性：12%
  - 结构与逻辑：12%
  - 技术/事实准确性：18%
  - 证据与个人贡献：18%
  - 结果与岗位匹配：10%
- 表达质量：30%
  - 清晰度：12%
  - 表达结构：10%
  - 节奏与简洁性：8%

## 评估原则

- 参考答案是多条合理路径，不是逐字标准答案。
- 每个判断必须引用回答文本或多模态报告中的明确证据。
- 面部、动作、视线和声音只允许描述可观察表现，不能直接推断心理状态。
- 缺少证据时返回 limitations，而不是补齐猜测。
